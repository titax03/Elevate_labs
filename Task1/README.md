Walmart Sales Dataset Cleaning and Preparation in Excel
Introduction

This project focuses on preparing the Walmart Sales dataset for analysis using Microsoft Excel. Data cleaning was performed to ensure consistency, accuracy, and usability of the dataset before conducting any business analysis or visualization.

Dataset Summary

Dataset: Walmart Sales Data

Software Used: Microsoft Excel

Records: 6,435

Variables: 8

Attributes Included
Store ID
Date
Weekly Sales
Holiday Indicator
Temperature
Fuel Price
Consumer Price Index (CPI)
Unemployment Rate
Data Quality Assessment
Missing Data Check

A review of all columns was conducted to identify blank or null values. No missing entries were detected in the dataset.

Duplicate Record Verification

Excel's duplicate-checking functionality was used to identify repeated records. The dataset contained no duplicate rows.

Data Formatting and Standardization

Several formatting improvements were applied to maintain consistency:

Dates were converted to a uniform date format.
Holiday indicators were transformed into descriptive values (Yes/No).
Numerical columns were rounded and displayed with two decimal places.
Weekly sales figures were formatted as financial values for improved readability.
